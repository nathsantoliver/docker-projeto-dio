# Projeto DIO Docker
Website criado para o projeto DIO (Criando um Container de uma aplicação WEB) utilizando um Servidor Nginx.

### Comandos para rodar a aplicação
##### Rodar (localhost:8000)
docker compose up -d

##### Verificar se o container está ativo
docker compose ps

##### Parar
docker compose stop

##### Remover o container
docker compose down
